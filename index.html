The Wondrous World of Wine

Beginner Trivia

Introduction:

This fun trivia game was designed to be a beginner level quiz that will test your basic knowledge in the wondrous world of wine.

Intructions:

  • There are 10 questions

  • You will get 2 minutes to answer all 10 questions

    • If you do not answer all the questions within the time allotted, the quiz will stop automatically and will reveal:

      1. The number of correct answers

      2. The number of incorrect answers

      3. The number of unanswered questions

    • If you answer all the questions within the time allotted, press the submit button to reveal:

      1. The number of correct answers

      2. The number of incorrect answers

      3. The number of unanswered questions

  • Press the start button below when you're ready to begin!
